Listas en HTML

En HTML tenemos tres tipos de listas

Listas no ordenadas

Las realizaremos con la etiqueta <ul>: unordered list

Dentro de cada lista tendremos al menos un elemento <li> list item

  • Primer elemento
  • Segundo
  • Tercero

No es recomendable pero podemos usar el atributo type

<ul type="disc" ><!-- "square" "circle" -->
	<li>Primer elemento</li>
	<li>Segundo</li>
	<li>Tercero</li>
</ul>

Listas ordenadas

Las realizaremos con la etiqueta <ol>: ordered list

Dentro de cada lista tendremos al menos un elemento <li> list item

  1. El uno
  2. El dos
  3. El tres

Listas anidadas

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Observa que la lista más interna esta totalmente contenida en u li


<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>
